## 📖 Description
The **AI Student Portal** is a high-performance educational ecosystem designed for the next generation of software engineers. It solves the problem of fragmented learning by providing a unified, distraction-free environment that integrates AI-powered code reviews, deep work synchronization, and gamified technical progression. This platform empowers students to master full-stack systems and agentic AI through a structured, high-fidelity experience.

## 🌐 Live Demo
- **URL:** [https://student-website-pearl.vercel.app/](https://student-website-pearl.vercel.app/)
- *Experience the full suite of AI-enhanced learning tools and technical roadmaps in action.*


## ✨ Features
- **🤖 AI Peer Reviewer:** Instant, line-by-line code reviews powered by agentic AI architects to ensure logic and quality.
- **⏱️ Deep Work Sync:** Integrated Pomodoro tracking and binaural beats to optimize focus and neural pathways.
- **🗺️ Technical Roadmaps:** Multi-phase journeys from fundamental architecture to building autonomous AI systems.
- **🛡️ Strict Room Protocol:** A distraction-free IDE environment with a 3-strike accountability system for peak productivity.
- **⚡ Student Sparks:** Gamified progression system where every commit and achievement earns you high-value Sparks.
- **🏗️ Skill-Locked Shop:** Redeem your earned Sparks for high-level technical resources and advanced modules.
- **📊 Performance Heatmaps:** Visual attendance and activity tracking to monitor consistency and learning velocity.
- **📑 Automated CV Generator:** Dynamically transform your portal achievements and skills into professional resumes.

## 🚀 Getting Started

### Prerequisites
- **Node.js 18+**
- **npm** or **yarn**
- **Git**

### Installation Steps
1. **Clone the repository:**
   ```bash
   git clone https://github.com/saad461/student-portal.git
   ```
2. **Navigate to the project folder:**
   ```bash
   cd student-portal
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Setup environment variables:**
   Create a `.env.local` file in the root directory (refer to the [Environment Variables](#-environment-variables) section).
5. **Run the development server:**
   ```bash
   npm run dev
   ```

## 🔑 Environment Variables
Create a `.env.local` file in the root directory and add the following variables:

```env
# Supabase Configuration
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key

# AI Provider Keys
GROQ_API_KEY=your_groq_api_key
MISTRAL_API_KEY=your_mistral_api_key
OPENROUTER_API_KEY=your_openrouter_api_key
GEMINI_API_KEY=your_gemini_api_key
```

## ☁️ Deployment
### Steps to deploy on Vercel:
1. **Push to GitHub:** Ensure your latest changes are pushed to your repository.
2. **Connect to Vercel:** Log in to Vercel and import your GitHub repository.
3. **Add Environment Variables:** Input all required variables in the Vercel project settings.
4. **Deploy:** Click the "Deploy" button. Vercel will handle the build and hosting.
