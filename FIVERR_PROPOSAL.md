## 🚀 Key Features

### 1. AI-Driven Technical Review
Students submit GitHub links, and the system automatically fetches the raw code. Admins use an AI-assisted tool to provide sectional grading and professional feedback in seconds.

### 2. Gamified Progression (Sparks & XP)
Keep students engaged with a points-based economy. Earn Sparks for completing "Boss Projects" and redeem them in a Skill-Locked Shop for exclusive resources.

### 3. Interactive Code Sandboxes
No more switching tabs. The platform includes built-in terminal-style code blocks and interactive sandboxes where students can test logic directly within the curriculum.

### 4. Advanced Curriculum Roadmap
A multi-phase "Technical Traverse" roadmap that automatically unlocks content based on student performance and prerequisites.

### 5. Automated Attendance & Streaks
The system tracks active portal usage and automatically marks attendance after a 15-minute threshold, building student consistency through streaks.

### 6. Career & Portfolio Center
Dynamic tracking of GitHub links, portfolio status, and AI-powered interview preparation sandboxes.

### 7. Performance Optimized
Engineered for speed with parallel data fetching, ensuring even the most data-heavy dashboards load in under 1.5 seconds.
